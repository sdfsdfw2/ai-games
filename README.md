# AI 游戏乐园 🎮

人工智能驱动的交互式模拟与游戏合集 — 纯前端，零依赖，开箱即玩。

## 在线预览

👉 https://sdfsdfw2.github.io/ai-games/

## 游戏列表

| 游戏 | 技术亮点 |
|---|---|
| 🐍 **AI 贪吃蛇** | BFS 寻路 + 控尾防御 + MATRIX 折叠算法 |
| ✈️ **AI 躲避子弹** | 729 状态空间强化学习，预判弹道 |
| 🧠 **AI 挡球** | 双 Q-learning AI 对抗，330 状态空间 |
| ⚔ **部落文明** | 神经网络自主决策，资源采集 / 战争 / 繁衍 |
| 🌍 **数字文明 · 进化生态模拟** | 食物链 + 城市聚落 + 昼夜循环 |
| 🧬 **神经网络进化模拟器** | 遗传算法 + 蜂群行为 |
| ⚛ **真实化学进化模拟器** | 基于真实元素周期表的化学涌现 |
| 🔮 **轨道弹球** | 二维圆周曲面碰撞物理，1x~15x 流速 |

## 特点

- 全部在浏览器中运行，无需服务器
- 涵盖强化学习、遗传算法、神经网络、物理模拟、化学涌现等多种 AI 技术
- 移动端适配，触屏支持
- 代码完全开源，可自由学习和修改

## 本地运行

直接用浏览器打开 `index.html` 即可，或通过任意静态服务器托管：

```bash
python3 -m http.server 8080
# 或
npx serve .
```
